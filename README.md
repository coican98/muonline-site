# Private MU Online Server Website

This project is a fully functional website developed using **PHP**, **Laravel**, and **JavaScript** for a **private MU Online server**. The website provides users with access to server-related content, including rankings, account management, and event information, along with integration for administrators to manage the server through the frontend.

## Layout Overview
The website features a **fixed header** with navigation, a **side menu** for user login and server details, and a **footer** with social links. Pages are designed to provide essential information to players while ensuring a responsive, visually appealing layout.

### Key Elements:
- **Header:** Includes the main navigation menu with links to the home, register, downloads, rankings, and account pages. The header updates dynamically when a user logs in.
- **Sidebar (Side Menu):** Contains a login box and server information, showing essential server details and event countdowns. For administrators, additional controls are visible.
- **Footer:** Displays social media links, including a **WhatsApp group** and a **Discord server**, ensuring the community stays connected.

## Pages

### Home Page
The homepage serves as the central hub for the website, displaying the following:
- **Castle Siege Information:** A banner prominently shows the **current guild** that owns the Castle and the **date of the next Castle Siege event**, providing players with vital competitive information.
- **Recent News Panel:** Displays the latest news, announcements, and events related to the server. These entries will eventually be part of the server’s **forum** section, keeping players informed of all updates.

# Site do Servidor Privado de MU Online

Este projeto é um site totalmente funcional desenvolvido usando **PHP**, **Laravel** e **JavaScript** para um **servidor privado de MU Online**. O site oferece aos usuários acesso a conteúdos relacionados ao servidor, incluindo rankings, gerenciamento de contas e informações de eventos, além de integração para que administradores gerenciem o servidor por meio do frontend.

## Visão Geral do Layout
O site possui um **cabeçalho fixo** com navegação, um **menu lateral** para login e detalhes do servidor, e um **rodapé** com links sociais. As páginas são projetadas para fornecer informações essenciais aos jogadores, garantindo um layout responsivo e visualmente atraente.

### Elementos Principais:
- **Cabeçalho:** Inclui o menu de navegação principal com links para as páginas inicial, cadastro, downloads, rankings e conta. O cabeçalho é atualizado dinamicamente quando um usuário faz login.
- **Menu Lateral:** Contém uma caixa de login e informações do servidor, mostrando detalhes essenciais e cronômetros de eventos. Para administradores, controles adicionais são visíveis.
- **Rodapé:** Exibe links para redes sociais, incluindo um **grupo no WhatsApp** e um **servidor no Discord**, garantindo que a comunidade permaneça conectada.

## Páginas

### Página Inicial
A página inicial serve como o hub central do site, exibindo o seguinte:
- **Informações do Castle Siege:** Um banner mostra de forma proeminente a **guild atual** que possui o castelo e a **data do próximo Castle Siege**, fornecendo informações competitivas vitais para os jogadores.
- **Painel de Notícias Recentes:** Exibe as últimas notícias, anúncios e eventos relacionados ao servidor. Estas entradas eventualmente farão parte da **seção de fórum** do servidor, mantendo os jogadores informados sobre todas as atualizações.

- **Código de usuário** (Um código único utilizado como uma segunda senha de autenticação dentro do jogo)
